
# FPS-Preview-Build
This is a demo build made in Unity of a simple FPS controller.

Version 0.1.8

## Controls:
```
WASD to move
Shift to run
Space to jump

Mouse1 to shot
Mouse2 to aim (only with sniper)

Q to change gun
E to throw a grenade
R to reload

F3 to show debug stats
Esc to exit to the menu
```
